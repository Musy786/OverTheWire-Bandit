## Problem
Read the only human readable file that is in the inhere directory.

## Solution 
cd inhere  
file ./*  
cat ./-file07  

## Explanation 
The "file" command allows us to see the file types of specified files. Paired with the `./*` command, which means
all files (`*`), including symbols in the file names ("/"), shows us all the file types of each file leaving "-file07"
as the only ASCII text file.
