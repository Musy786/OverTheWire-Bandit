## Problem
Read the base64 encrypted data.txt file.

## Solution 
base64 -d data.txt

## Explanation 
The text file is encrypted with base64 so the "-d" allows us to decode the text file to find the password.

## Password 
dtR173fZKb0RRsDFSGsg2RWnpNVj3qRr

## Lesson Learnt
Decrypting a file can be done quite easily if the encryption method is known.
